# Roshan Academy Quiz App 🎓

An interactive web-based quiz application tailored for **Maharashtra State Board** students. This app provides a fun and engaging way for students from **1st to 10th Standard** to test their knowledge across various subjects.

## 🚀 Features
* **Grade-Specific Content:** Dynamically loads subjects and questions based on the selected standard (1st - 10th).
* **Timed Challenges:** 30-second countdown per question to encourage quick thinking.
* **Interactive UI:** Built with **Tailwind CSS** for a responsive, modern look.
* **Progress Tracking:** Visual progress bar and instant feedback on correct/incorrect answers.
* **Achievements:** Earn Gold, Silver, or Bronze medals based on your performance.
* **Persistent High Scores:** Saves your best scores locally so you can challenge yourself later.

## 🛠️ Tech Stack
* **HTML5** & **CSS3**
* **Tailwind CSS** (Styling)
* **JavaScript (ES6+)** (Logic & Dynamic Data Mapping)
* **FontAwesome** (Icons)

## 📂 Project Structure
* `index.html` - The main UI and quiz logic.
* `quizData.js` - Centralized database for all standards and subjects.
* `images/` - Assets for backgrounds and favicons.

## 🔧 How to Use
1. Clone the repository.
2. Open `index.html` in any modern web browser.
3. Select your Standard and Subject to begin!
